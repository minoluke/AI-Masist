# Title: Threshold Public Goods Game (TPGG) - LLMエージェントによるルール遵守と協力行動

## Keywords
threshold public goods game, rule compliance, cooperation, LLM agents, social dilemma, collective action

## TL;DR
グループに提示される拠出ルールの性質（「ちょうど必要な額」vs「多めの要求」）が、LLMエージェントの協力行動・ルール遵守・効率にどう影響するかを調査する。

## Abstract
公共財ゲームにおける協力の創発と維持は、社会シミュレーションの中心的な研究課題である。本研究では、しきい値公共財ゲーム（TPGG）を用いて、グループに提示される「拠出ルール」の性質がLLMエージェントの行動にどのような影響を与えるかを調査する。

具体的には、4人グループで一定額のトークンを共同拠出し、合計がしきい値を超えればご褒美が得られるゲームにおいて、以下の2種類のルールを比較する：
- **必要額ちょうどのルール**: しきい値達成に必要な合計額をちょうど要求
- **多めのルール**: 必要額より多い合計を要求

研究質問：
- 多めに要求されたルールは、行動の乱れ（ルール違反）を増やすか？
- 必要分ちょうどのルールは、安定した協力を生むか？
- グループ全体の効率（過剰拠出の有無）はルールによってどう変わるか？

仮説：
1. 多めに要求されたルールは守る人が減りやすい（協調性の低下）
2. 目標達成率への影響はルールの違いではあまり変わらない
3. 多めのルールは「出しすぎ（無駄）」を増やし、効率を下げる
4. 1人あたりの負担がピッタリ均等割りできる場合、協力がまとまりやすい

本研究は、LLMエージェントを用いた社会シミュレーションにおいて、制度設計（ルールの提示方法）が協力行動に与える影響を定量的に評価することで、マルチエージェントシステムにおける規範設計への示唆を提供する。
