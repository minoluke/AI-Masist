# Title: 「どうでもいい多数決」における説得・同調・焦点形成（LLMエージェント）

## Keywords
trivial decision-making, majority vote, persuasion, conformity, focal point, framing, default effect, sequential voting, social influence, LLM agents

## TL;DR
重要性の低い二択の多数決では、説得（好みの変化）よりも、手続き設計や情報提示による同調・焦点形成が投票結果を左右する。LLMエージェントを用いて、そのメカニズムを分解・定量化する。

## Abstract
本研究は、「猫を“ぴゃおぴゃお”と呼ぶか“ひゃかひゃか”と呼ぶか」といった実質的にどうでもよい低関与な二択を多数決で決める状況において、どのような条件下で集団が特定の選択肢に収束するのかを検討する。  
低関与な争点では、個々の選好は固定されておらず、フレーミング、投票の公開性や逐次性、途中経過の提示、デフォルト設定といった制度的要因によって容易に変動・再構成される可能性がある。  

本研究では、LLMエージェント集団に弱い初期選好と協調インセンティブを与え、同時投票・逐次投票、匿名・公開、理由提示の有無などの条件を操作することで、(i) 内的選好の変化としての「説得」と、(ii) 好みとは独立した「協調的同調（焦点形成）」を区別して測定する。  
その結果、低関与多数決においては価値観の変容よりも、社会的証明や情報カスケードによる焦点化が支配的であること、そして短く伝播しやすい理由やデフォルト設定が集団収束を強く促進することが示唆される。  

本研究は、LLMを用いた社会シミュレーションを通じて、「説得が不要なときに何が集団意思決定を動かしているのか」という基礎的だが見過ごされがちな問題を明らかにし、制度設計と規範形成の理解に貢献する。
